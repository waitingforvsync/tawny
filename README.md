# Tawny

A BBC Micro emulator written in Rust.

Named after the tawny owl — the BBC Micro logo is a stylised owl made from dots.

## Status

Early development. Currently opens a window with a wgpu-rendered surface.

## Building

```sh
cargo build
cargo run
```

## Tech stack

- **Rust** (edition 2024)
- **winit** — windowing and event loop
- **wgpu** — GPU rendering
